# Alhilli-Supermarket-Dashboard
An Excel dashboard project for analyzing supermarket sales, profit, and units sold across branches, categories, and months.


## 📌 Project Overview
The dashboard presents key business insights related to:
- Profit by branch and category
- Units sold by month
- Profit trends by month

The goal of this project is to transform raw supermarket data into clear and useful visual insights that support better understanding and decision-making.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Dashboard Design

## 📈 Key Insights Covered
- Comparison of branch performance
- Category-level profit analysis
- Monthly sales and profit trends
- Summary visualizations through an interactive dashboard

## 💡 Skills Demonstrated
- Data Cleaning
- Data Analysis
- Data Visualization
- Dashboard Creation
- Business Reporting

## 🎯 Project Goal
To build a clear and professional dashboard that helps analyze supermarket performance in an easy and visual way.
